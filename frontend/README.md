# Pomotoro

## Development

### Prerequisites

- Bun.js
- Rustup
- Follow [tauri prerequisites](https://v2.tauri.app/start/prerequisites/)

### Installing dependencies

```
bun install
```

### Running debug app

```
bun tauri dev
```

## Building

```
bun tauri build
```

Then check app in src-tauri/target/release
